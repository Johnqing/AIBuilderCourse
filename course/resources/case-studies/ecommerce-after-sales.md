# 电商售后工单自动化

> 工单系统 → Webhook触发 → OpenClaw智能处理 → 自动退款/换货

---

## 案例背景

| 维度 | 信息 |
|------|------|
| 行业 | 电商（服装类） |
| 规模 | 中型店铺，月均工单 5000+，客服 8人 |
| 痛点 | 售后工单积压、处理慢、退款周期长 |
| 目标 | 工单自动分类 + 退款/换货自动化，让人效提升60% |

---

## 业务痛点分析

### 售后工单分布

| 问题类型 | 占比 | 典型问题 |
|---------|------|---------|
| 退货退款 | 40% | 不喜欢/尺码问题/质量问题 |
| 换货 | 25% | 换颜色/换尺码/换款式 |
| 物流问题 | 20% | 未发货/物流延迟/丢件 |
| 其他 | 15% | 发票/改地址/咨询 |

### 原有工单流程问题

```
用户提交工单 → 人工判断类型 → 查找订单 → 处理退款/换货 → 确认完成
                    ↓
              平均处理时间 15分钟+
```

---

## 解决方案：售后工单自动化工作流

### 整体架构

```
用户提交工单（多渠道：App/电话/微信）
    ↓
工单系统创建工单
    ↓
Webhook触发OpenClaw
    ↓
┌─────────────────────────────────────┐
│     OpenClaw 售后工单处理引擎          │
│                                      │
│  1. 工单预分析（AI提取订单+问题）       │
│  2. 自动分类（退货/换货/物流/其他）     │
│  3. 订单核实（调用订单API）           │
│  4. 自动处理（符合条件直接处理）        │
│  5. 复杂问题转人工                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│         处理结果                      │
│  符合条件 → 自动退款/换货/物流查询    │
│  不符合条件 → 发送处理方案给用户确认   │
│  复杂问题 → 转人工处理               │
└─────────────────────────────────────┘
```

---

## 核心流程详解

### Step 1：工单创建 + Webhook触发

```json
{
  "ticket_id": "AF-20260409-001",
  "source": "app",
  "user_id": "user_12345",
  "order_id": "T-20260401-88888",
  "type": "退货退款",
  "description": "尺码不合适，想退货",
  "amount": 299,
  "reason": "尺码问题"
}
```

### Step 2：OpenClaw自动处理规则

```yaml
处理规则：

退货退款（金额≤500）：
  - 订单签收7天内 → 自动通过 → 直接退款
  - 订单签收7-15天 → 需用户上传图片 → 审核后退款
  - 订单签收15天以上 → 转人工处理

换货：
  - 同款有货 → 生成换货单 → 通知用户寄回
  - 同款无货 → 转人工处理

物流问题：
  - 未发货 → 直接拦截 → 通知用户
  - 物流延迟 → 查询物流 → 告知用户
  - 丢件 → 转人工处理 → 补发或退款
```

### Step 3：自动处理 vs 转人工

```
自动处理（80%）：
  - 退货退款（签收7天内）
  - 换货（同款有货）
  - 物流查询
  - 未发货拦截

转人工处理（20%）：
  - 退货退款（签收15天以上）
  - 换货（同款无货）
  - 丢件
  - 用户投诉
  - 金额异常
```

## 落地效果

### 量化指标

| 指标 | 上线前 | 上线后 | 提升 |
|------|-------|-------|------|
| 平均处理时间 | 15分钟 | 2分钟 | 87%↓ |
| 自动处理率 | 0% | 80% | +80% |
| 客服人效 | 30件/人/天 | 80件/人/天 | 167%↑ |
| 退款周期 | 3-5天 | 即时 | 90%↓ |
| 客诉率 | 8% | 3% | 62%↓ |

### ROI

```
月均工单: 5000
人效提升: 167%
节省人力: 5000 ÷ 80 × 167% ≈ 10人

人力成本节省: 10人 × 6000元/月 = 60000元/月
退款周期缩短减少客诉: 约20000元/月

月总收益: ~80000元
投入回收期: ~3个月
```
