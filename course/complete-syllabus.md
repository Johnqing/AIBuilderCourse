# 90天AI Builder实战营
## 完整课程大纲

> 从产品能力 → Vibe Coding → 商业落地

---

## 课程概览

| 维度 | 内容 |
|------|------|
| 课程名称 | AI Builder实战营 |
| 总时长 | 90天（12周） |
| 目标学员 | 企业内部团队（产品/运营/技术） |
| 先决条件 | 零基础可学 |
| 核心目标 | 掌握AI Builder落地能力，独立完成1个真实业务场景 |

## 课程模块详解

### M0 预备模块（可选）

**对象**：零基础学员

**时长**：3周

**内容**：
- AI认知：什么是AI、AI能做什么不能做什么
- 对话能力：有效提问、追问、批判对话
- 编程入门：编程基本概念、AI辅助编程体验

**考核**：入学测试 + 跟练项目

---

### M1 AI Builder基础

**时长**：2周（Week 1-2）

**学习目标**：掌握AI Builder核心能力

| 课次 | 主题 | 时长 |
|------|------|------|
| M1-1 | 产品架构入门 | 60min |
| M1-2 | 技能体系 | 60min |
| M1-3 | 节点与路由 | 60min |
| M1-4 | 定时与触发 | 60min |
| M1-5 | 消息通道 | 60min |
| M1-6-10 | 综合练习+考核 | 5天 |

**输出物**：AI Builder架构图

**认证**：L1认证（AI Builder操作认证）

---

### M2 Vibe Coding

**时长**：4周（Week 3-6）

**学习目标**：用AI辅助编程，快速构建原型

| 课次 | 主题 | 时长 |
|------|------|------|
| M2-1 | AI编程思维 | 60min |
| M2-2 | Codex实战入门 | 90min |
| M2-3 | Codex进阶 | 90min |
| M2-4 | Claude Code入门 | 90min |
| M2-5 | 前端快速搭建 | 120min |
| M2-6 | 后端与API集成 | 120min |
| M2-7-8 | 综合项目+答辩 | 5天 |

**输出物**：可运行的项目原型

**认证**：L2认证（Vibe Coding能力认证）

---

### M3 场景设计

**时长**：2周（Week 7-8）

**学习目标**：挖掘业务场景，设计落地方案

| 课次 | 主题 | 时长 |
|------|------|------|
| M3-1 | 业务挖掘 | 60min |
| M3-2 | 需求分析 | 60min |
| M3-3 | 方案设计 | 90min |
| M3-4 | 成本测算 | 60min |
| M3-5 | 方案评审 | 3天 |

**输出物**：商业落地方案文档

**认证**：L3认证（场景设计能力认证）

---

### M4 商业落地

**时长**：2周（Week 9-10）

**学习目标**：将AI能力落地到真实业务

| 课次 | 主题 | 时长 |
|------|------|------|
| M4-1 | AI客服落地 | 90min |
| M4-2 | 内容自动化 | 90min |
| M4-3 | 销售自动化 | 90min |
| M4-4 | 项目管理自动化 | 90min |
| M4-5 | 案例复盘 | 120min |

**输出物**：MVP定义文档

---

### M5 实战项目

**时长**：2周（Week 11-12）

**学习目标**：真实场景从0到1落地

| 阶段 | 内容 |
|------|------|
| Day 78-80 | 方案打磨 |
| Day 81-85 | 开发冲刺 |
| Day 86-88 | 内部测试 |
| Day 89-90 | 答辩路演 |

**输出物**：
- AI工作流配置
- 项目演示Demo
- 项目复盘报告

**认证**：L4认证（AI Builder实战认证）

## 考核体系

### 认证等级

| 等级 | 要求 | 认证内容 |
|------|------|---------|
| L1 | M1+考试≥80分 | AI Builder操作认证 |
| L2 | M2原型+答辩 | Vibe Coding能力认证 |
| L3 | M3+M4方案评审 | 场景设计能力认证 |
| L4 | 90天答辩+上线 | AI Builder实战认证 |

### 过程考核

| 维度 | 占比 | 考核方式 |
|------|------|---------|
| 输出物 | 40% | 作业、文档、项目 |
| 能力认证 | 30% | L1-L4认证 |
| 过程表现 | 20% | 每日参与、答疑 |
| 团队协作 | 10% | 组长评价+互评 |
