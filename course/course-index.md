# AI Builder 实战营

> 从产品能力 → Vibe Coding → 商业落地

---

## 📌 课程概述

| 维度 | 内容 |
|------|------|
| 课程名称 | AI Builder实战营 |
| 课程时长 | 90天（企业内训版）/ 126天（零基础版） |
| 目标学员 | 企业内部团队（产品/运营/技术） |
| 核心目标 | 掌握AI Builder落地能力，独立完成1个真实业务场景 |

---

## 🎯 能力模型

```
AI Builder能力 = 产品认知 × Vibe Coding × 商业落地
                     ↓            ↓            ↓
                  理解能力      实现能力       价值能力
```

---

## 📐 整体架构

```
Week 1-2：认知奠基（产品能力篇）
Week 3-6：产品构建（Vibe Coding篇）
Week 7-10：场景落地（商业化篇）
Week 11-12：实战答辩与上线
```

## 🎓 考核体系

| 等级 | 要求 | 认证 |
|------|------|------|
| L1 基础 | M1完成+考试≥80分 | AI Builder操作认证 |
| L2 进阶 | M2原型产出+答辩 | Vibe Coding能力认证 |
| L3 落地 | M3+M4方案评审 | 场景设计能力认证 |
| L4 全栈 | 90天答辩+方案上线 | AI Builder实战认证 |

---

## 📦 量化成果指标

| 指标 | 目标 |
|------|------|
| 每组产出物 | 至少1个可演示AI工作流 |
| 每组落地场景 | 至少1个真实业务场景MVP |
| 效率提升 | 试点场景人力节省≥30% |
| 方案通过率 | 方案评审通过率≥80% |
| L2认证获取率 | ≥70% |
