# AI Builder 实战营

> 从提示词工程 → PM×AI → Vibe Coding → 商业落地 → 一人公司

---

## 📌 课程概述

| 维度 | 内容 |
|------|------|
| 课程名称 | AI Builder实战营 |
| 课程时长 | 90天（企业内训版）/ 30天（压缩版） |
| 目标学员 | 企业内部团队（产品/运营/技术）/ 有志于一人公司的创业者 |
| 核心目标 | 成为能独立完成产品设计和落地的Builder |

---

## 🎯 Builder 能力模型

```
Builder = 提示词工程 × PM综合能力 × Vibe Coding × 商业落地
               ↓            ↓            ↓             ↓
            M0核心       M1核心       M2核心        M3-M5核心
```

---

## 📐 整体架构

```
Week 0-1：提示词工程（M0）
Week 1-2：PM×AI（M1）
Week 3-6：Vibe Coding（M2）
Week 7-8：业务分析与方案设计（M3）
Week 9-10：技术实施与商业落地（M4）
Week 11-12：实战项目与一人公司验证（M5）
```

## 🎓 认证体系（嵌入式考核）

认证融入授课过程，由讲师在教学过程中完成评估。

| 认证 | 要求 | 对应模块 |
|------|------|---------|
| L1 操作认证 | M1考核通过 | AI Builder操作 |
| L2 编程认证 | M2考核通过 | Vibe Coding |
| L3 方案认证 | M3考核通过 | 场景设计 |
| L4 技术认证 | M4+M5考核通过 | 技术实战 |
| L5 一人公司认证 | 商业落地项目 | 一人公司 |

### 毕业路径

| 路径 | 要求 | 适合人群 |
|------|------|---------|
| 技术路径 | L1+L2+L3+L4 | 想成为技术Builder |
| 综合路径 | L1+L2+L3+L4+L5 | 想成为一人公司创始人 |

## 📅 班型适配

| 班型 | 人数 | 项目形式 | 考核特点 |
|------|------|---------|---------|
| 微班 | 1-2人 | 个人项目 | 讲师1对1辅导，个人答辩 |
