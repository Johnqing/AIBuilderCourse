# 毕业后支持体系

## 一、毕业条件

### 1. 必须完成

| 要求 | 说明 |
|------|------|
| 全勤率 | ≥80%出勤 |
| 作业完成 | 100%提交 |
| L1认证 | 必须通过 |
| 毕业答辩 | 必须参加 |

### 1.2 可选完成

| 要求 | 说明 |
|------|------|
| L2认证 | 推荐通过 |
| L3认证 | 推荐通过 |
| L4认证 | 挑战目标 |

## 二、毕业后支持

### 2.1 社群支持

| 社群类型 | 加入条件 | 服务内容 |
|---------|---------|---------|
| 毕业生社群 | 毕业即可加入 | 答疑、分享 |
| L4精英社群 | L4认证通过 | 深度交流、跨界合作 |
| 导师社群 | 表现优秀者 | 优先答疑、资源对接 |

### 2.2 答疑支持

```
毕业后30天内：
- 课程相关问题：免费答疑
- 落地相关问题：3次免费咨询

毕业后30天后：
- 课程相关问题：社群答疑
- 落地相关问题：付费咨询
```

### 2.3 资源支持

毕业生可获得：
- 最新课程更新
- 新增案例分享
- 行业报告订阅
- 工具使用优惠

## 三、落地支援

### 3.1 落地困难识别

| 困难类型 | 表现 | 支援方式 |
|---------|------|---------|
| 技术困难 | 工作流配置问题 | 远程协助 |
| 资源困难 | 缺少API权限 | 对接企业 |
| 业务困难 | 场景设计不清晰 | 方案评审 |
| 组织困难 | 领导不支持 | 沟通材料支持 |

### 3.2 支援申请

```
申请流程：

1. 在社群内说明困难
2. 导师评估困难类型
3. 匹配支援资源
4. 协助解决
```

## 四、持续学习

### 4.1 进阶课程

毕业生可继续学习：
- 高级AI应用课程
- 行业专题课程
- 案例拆解课程

### 4.2 校友活动

- 每月线上分享会
- 每季度线下聚会
- 每年校友大会

## 五、推荐与激励机制

### 5.1 推荐奖励

| 推荐行为 | 奖励 |
|---------|------|
| 推荐新学员 | 课程优惠码/现金 |
| 推荐企业内训 | 佣金分成 |
| 推荐案例入库 | 署名+奖励 |

### 5.2 优秀学员

毕业后表现优秀者可：
- 成为预备讲师
- 参与课程开发
- 获得更多曝光机会
