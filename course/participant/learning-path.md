# 学员学习路径与个人收益

## 一、学习路径总览

```
入口：入学测试 → 确定学习路径
    ↓
┌─────────────────────────────────────────┐
│  学习路径选择                              │
├─────────────────────────────────────────┤
│  路径A：执行者路径（运营/人）              │
│  路径B：管理者路径（PM/总监）              │
│  路径C：创业者路径（商业落地）             │
└─────────────────────────────────────────┘
    ↓
90天学习 + L1-L4认证
    ↓
终局：毕业认证 + 个人作品集
```

## 二、学员个人收益体系

### 2.1 能力收益

学完90天，学员能：

| 能力层级 | 具体能力 | 可胜任岗位 |
|---------|---------|-----------|
| L1 基础 | AI Builder基础操作 | AI工具使用者 |
| L2 进阶 | Vibe Coding | AI应用开发者 |
| L3 落地 | 场景设计与方案 | AI项目负责人 |
| L4 专家 | 独立落地AI项目 | AI转型负责人 |

### 2.2 认证收益

| 认证等级 | 证书名称 | 市场价值 |
|---------|---------|---------|
| L1 | AI Builder操作师 | 基础认证 |
| L2 | AI应用开发师 | 中级认证 |
| L3 | AI场景设计师 | 高级认证 |
| L4 | AI实战专家 | 专家认证 |

### 2.3 职场收益

- **薪资提升**：据市场数据，AI技能者薪资溢价20-40%
- **职位竞争力**：AI项目经验优先录用
- **内部晋升**：AI转型负责人储备人才

## 三、学习路径详解

### 路径A：执行者路径（运营/人）

**适合人群**：运营、人、行政等执行岗

**学习重点**：
- AI工具使用
- 日常工作自动化
- 内容生成

**输出物**：AI辅助工作手册

---

### 路径B：管理者路径（PM/总监）

**适合人群**：产品经理、项目总监等管理岗

**学习重点**：
- AI项目管理
- 需求分析与拆解
- 团队AI培训

**输出物**：AI团队管理方案

---

### 路径C：创业者路径（商业落地）

**适合人群**：创业者、业务负责人

**学习重点**：
- AI商业化落地
- 成本与ROI
- MVP验证

**输出物**：AI商业落地方案

## 四、个人作品集

每位学员毕业时需产出：

1. **AI工作流作品集**（3个以上）
2. **商业落地方案**（1份）
3. **学习复盘报告**（1份）

这些作品可：
- 放入个人简历
- 用于内部述职
- 作为跳槽面试素材
