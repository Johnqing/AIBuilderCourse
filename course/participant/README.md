## 🎯 学习目标

完成本模块后，你应该能够：
- [ ] 说出AI Builder在OpenClaw中的定位
- [ ] 理解技能体系和工作流概念
- [ ] 能独立配置简单的工作流
- [ ] 能接入1-2个消息通道

| L2 | M2原型产出+答辩 | Vibe Coding能力认证 |

## 💡 学习建议

1. **预习**：课前观看录播，带着问题来上课
2. **动手**：不要只看，要实际操练
3. **提问**：遇到问题及时在群里问
4. **复习**：课后整理笔记，形成自己的知识体系
5. **练习**：用真实业务场景练习，学以致用
