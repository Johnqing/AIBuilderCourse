# M5 实战项目

> Week 11-12：真实场景从0到1落地，答辩验收

---

## 📌 章节目标

| 目标 | 说明 |
|------|------|
| 核心目标 | 每个小组落地1个真实业务场景 |
| 认证目标 | 通过L4认证（AI Builder实战认证） |
| 交付目标 | 可演示、可量化、可复用 |

## 📖 项目节奏

### Day 78-80：方案打磨

| 环节 | 内容 |
|------|------|
| 导师辅导 | 一对一方案优化 |
| 方案定稿 | 确认最终方案 |
| 资源配置 | 确认所需资源 |

### Day 81-85：开发冲刺

| 环节 | 内容 |
|------|------|
| 任务分解 | 按方案分解开发任务 |
| 每日站会 | 同步进度、解决问题 |
| 核心功能开发 | 优先实现核心闭环 |

### Day 86-88：内部测试

| 环节 | 内容 |
|------|------|
| 交叉测试 | 跨组交叉测试 |
| 问题修复 | 发现问题及时修复 |
| 体验优化 | 细节打磨 |

### Day 89-90：答辩路演

| 环节 | 内容 |
|------|------|
| 成果展示 | 每组15分钟展示 |
| 评委点评 | 评审团打分 |
| 评优颁奖 | 优秀方案表彰 |
| 方案归档 | 优秀方案入库 |
