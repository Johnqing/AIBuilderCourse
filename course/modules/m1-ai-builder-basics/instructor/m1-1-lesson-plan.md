## 教学流程

### 一、导入（5分钟）

**开场问题**：
> "大家平时用AI工具吗？用AI做什么？"

**引出痛点**：
- AI工具越来越多，不知道怎么串联起来
- AI能力分散在不同平台，管理成本高
- 没有统一的方式把AI能力落地到业务

**引出解决方案**：
- OpenClaw AI Builder = AI能力的「操作系统」

---

### 二、理论讲解（20分钟）

#### 2.1 什么是AI Builder

**定义**：
AI Builder是OpenClaw中的AI能力编排平台，让用户通过可视化配置，把各种AI能力串联成工作流。

**类比理解**：
```
传统开发：代码 → API → 服务
AI Builder：节点 → 工作流 → AI能力
```

#### 2.2 OpenClaw架构

```
┌─────────────────────────────────────────────────┐
│                   OpenClaw                       │
│                                                  │
│  ┌─────────────┐    ┌─────────────────────┐   │
│  │   消息通道    │    │      AI Builder     │   │
│  │  飞书/企微   │    │  ┌───────────────┐ │   │
│  │  钉钉/Slack │    │  │   技能市场      │ │   │
│  └─────────────┘    │  │   技能编排      │ │   │
│                      │  │   工作流配置    │ │   │
│  ┌─────────────┐    │  │   节点管理      │ │   │
│  │   定时触发   │    │  └───────────────┘ │   │
│  │  事件触发    │    └─────────────────────┘   │
│  │  Webhook   │                               │
│  └─────────────┘                               │
└─────────────────────────────────────────────────┘
```

#### 2.3 核心模块

| 模块 | 作用 | 类比 |
|------|------|------|
| 技能 | AI能力的原子单元 | 乐高零件 |
| 工作流 | 多个技能的编排 | 乐高图纸 |
| 节点 | 工作流中的处理单元 | 流水线上的工位 |
| 通道 | 消息的入口和出口 | 流水线进出口 |

#### 2.4 AI Builder能做什么

| 场景 | 示例 |
|------|------|
| 客服自动化 | 智能客服 + 工单分发 |
| 内容自动化 | 选题 → 生成 → 发布 |
| 销售自动化 | 线索打分 → 跟进提醒 |
| 管理自动化 | 任务分配 → 进度追踪 |

#### 2.5 什么时候用AI Builder

**适合的场景**：
- 需要连接多个AI能力
- 需要根据条件做判断分支
- 需要定时或事件触发
- 需要跨平台消息路由

**不适合的场景**：
- 简单的一次性问答（直接用ChatGPT）
- 纯数据处理不需要AI（用脚本）

### 三、演示（10分钟）

**演示内容**：打开OpenClaw界面，展示：
1. AI Builder入口位置
2. 技能市场的样子
3. 创建一个简单工作流
4. 添加一个节点

**讲解要点**：
- 绿色图标是触发节点
- 蓝色图标是处理节点
- 紫色图标是输出节点

### 四、跟练（30分钟）

#### 跟练任务1：熟悉界面（10分钟）

```
任务：
1. 登录OpenClaw
2. 找到AI Builder入口
3. 浏览技能市场，记录3个感兴趣的技能
4. 截图发到班级群
```

#### 跟练任务2：创建第一个工作流（15分钟）

```
任务：
1. 创建一个名为「test-你的姓名」的工作流
2. 添加一个「消息入口」节点
3. 添加一个「AI回复」节点
4. 连接两个节点
5. 点击保存
6. 截图发到班级群
```

#### 跟练任务3：绘制架构图（5分钟）

```
任务：
手绘OpenClaw架构图，包含：
- 消息通道层
- AI Builder层
- 核心模块
- 至少2个技能示例

完成后拍照发到班级群
```

### 五、总结（5分钟）

**今日要点**：
1. AI Builder是AI能力的编排平台
2. 核心概念：技能、工作流、节点、通道
3. 适合场景：多AI能力串联、条件分支、定时触发

**课后作业**：
1. 复习今日内容
2. 预习M1-2：技能体系
3. 思考：你的工作中有什么场景可以用AI Builder？
