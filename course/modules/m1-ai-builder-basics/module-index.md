# M1 AI Builder基础

> Week 1-2：产品经理综合能力提升——用 AI Builder 提升产品经理的核心工作效能
> **核心主题**：产品经理的综合能力——需求分析、方案设计、项目管理、数据汇报
> **工具支撑**：用 AI Builder 把这些能力从「人工」变成「AI辅助」

---

## 本模块核心目标

```
产品经理的核心工作：
需求分析 → 方案设计 → 项目管理 → 数据汇报

每个环节都有大量重复性工作，可以交给AI：
- 访谈记录AI总结 → 人做判断
- PRD框架AI生成 → 人做决策
- 周报AI自动汇总 → 人做分析
- 数据异常AI告警 → 人做应对

学完M1，你将能：
- 用AI辅助需求分析（记录→整理→优先级建议）
- 用AI辅助方案设计（写PRD、写文档）
- 用AI辅助项目管理（定时提醒、自动汇总、风险预警）
- 用AI辅助数据汇报（日报自动生成、异常告警）
- 用AI Builder把日常工作流自动化
```
