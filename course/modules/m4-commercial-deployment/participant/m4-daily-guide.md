> **核心主题**：技术实施能力——把M3设计的方案，用OpenClaw配置出来

## M4 vs M3 的区别

```
M3：设计方案（人做的事）
- AI场景识别
- 需求分析
- ROI测算

M4：技术实现（OpenClaw配置）
- 把M3方案配置出来
- 测试验证闭环
```

## Week 9

### Day 1：AI客服工作流配置

**今日目标**
- [ ] 掌握意图识别节点配置

```
4. 配置知识库检索
5. 配置路由分支（3个置信度档位）
6. 配置AI回复节点
7. 配置转人工节点
8. 手动触发，测试闭环

独立练习（下午）：
完善AI客服工作流：
1. 添加更多意图（至少6个）
2. 完善知识库内容
3. 测试不同置信度的分支
```

## 常见问题 FAQ

| 问题 | 答案 |
|------|------|
| 意图识别不准 | 增加训练数据，每个意图至少10条 |
| 知识库没内容 | 先用AI生成模拟内容，后续人工维护 |
| Webhook接不上 | 用手动触发测试工作流本身，Webhook单独调试 |
| 分支测试不触发 | 检查条件表达式是否正确 |
