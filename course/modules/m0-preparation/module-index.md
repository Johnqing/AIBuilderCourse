# M0 预备模块：提示词工程核心

> 从「会用AI」升级到「用好AI」——结构化Prompt、错误预判、批判对话

---

## 定位说明

```
M0假设：学员已会基本AI操作（注册、会话、复制粘贴）

M0目标：把学员从「会用」提升到「用好」

核心差距：
会用 = 问一个问题，得到一个答案
用好 = 用正确的方式问，得到可用的答案
```

---

## 核心能力

| 能力 | 说明 |
|------|------|
| 结构化Prompt | 角色+任务+要求，一次做对 |
| 错误预判 | 在Prompt里堵住漏洞 |
| 批判对话 | 不盲目接受，持续追问优化 |
| 工作流模板 | 把Prompt模板化，提升效率 |
