# M2 Vibe Coding

> Week 3-6：用AI辅助编程，快速构建产品原型
> **核心主题**：工具使用（L1）→ 调试能力（L2）→ 工程化能力（L3）→ 综合能力（L4），逐步递进

---

## 本模块核心目标

```
vibe coding 能力 = 工具使用 × 调试能力 × 工程化能力

L1 工具使用：
→ 会用 Codex/Claude Code 生成代码
→ Prompt 写得好，代码质量高

L2 调试能力：
→ 代码出问题，能定位问题
→ 能引导 AI 正确修复

L3 工程化：
→ 代码可维护、可扩展、可协作
→ 能做前后端对接，能做代码重构

L4 综合：
→ 能独立完成完整项目

学完 M2，你将能：
- 用 AI 高效生成代码（工具使用）
- 遇到问题能独立调试（调试能力）
- 写出可维护的代码（工程化能力）
- 完成一个完整项目（综合能力）
```

## 📖 课程内容

### L1：工具使用（Week 3）

#### M2-1：AI编程思维 + Codex入门

| 维度 | 内容 |
|------|------|
| 主题 | vibe coding 本质、Prompt 质量决定代码质量 |
| 核心 | 结构化 Prompt，让代码一次做对 |

#### M2-2：Codex进阶 + 错误预判

| 维度 | 内容 |
|------|------|
| 主题 | 错误预判三步法、防错 Prompt |
| 核心 | 在问题发生前，在 Prompt 里堵住漏洞 |

### L2：调试能力（Week 4）

#### M2-3：代码调试：定位问题

| 维度 | 内容 |
|------|------|
| 主题 | 读懂报错、分析问题、让 AI 定位 |
| 核心 | 调试第一步：读懂报错 |

#### M2-4：代码调试：修复问题

| 维度 | 内容 |
|------|------|
| 主题 | 一次修一个、逐个验证、多轮追问 |
| 核心 | 调试第二步：正确引导 AI 修复 |

### L3：工程化能力（Week 5）

#### M2-5：代码重构与代码质量

| 维度 | 内容 |
|------|------|
| 主题 | AI 重构、可读性、性能优化 |
| 核心 | 让代码从「能跑」变成「跑得好」 |

#### M2-6：项目集成与接口设计

| 维度 | 内容 |
|------|------|
| 主题 | 前后端对接、接口规范、格式对齐 |
| 核心 | 工程化最后一关：可协作 |

### L4：综合能力（Week 6）

#### M2-7：综合项目

| 维度 | 内容 |
|------|------|
| 主题 | 完整项目，综合运用 L1-L3 |
| 核心 | 真实项目经历，完整能力闭环 |

#### M2-8：项目答辩

| 维度 | 内容 |
|------|------|
| 主题 | 展示问题解决过程 |
| 核心 | 不是展示成果多漂亮，而是展示调试能力 |
