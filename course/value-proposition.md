# 课程价值主张

> 不教你「什么是AI」，教你「用AI做出东西」

---

## 企业培训价值

### 老板关心什么？

| 老板的痛点 | 我们能解决 |
|-----------|-----------|
| 花了钱培训，但看不到效果 | **90天后有可演示的成果**，不是听课记录 |
| 学会了但落不了地 | **用自己公司的真实业务场景**做练习 |
| 培训完人跑了 | **能力认证体系**，留住有证书的人才 |
| 不知道员工学得怎么样 | **L1-L4等级认证**，可量化考核 |

### 学员关心什么？

| 学员的担心 | 我们能解决 |
|-----------|-----------|
| 零基础怕听不懂 | **50%理论先打基础**，先学后做 |
| 太难了跟不上 | **讲师带练环节**，每步跟做 |
| 学完还是不会用 | **循序渐进的练习**，从简单到复杂 |
| 学了用不上 | **作业就是真实工作**，学了就能用 |

---

## 学员能得到什么？

### 能力层面

```
学完90天，你能：
├── 用AI Builder配置智能客服
├── 用AI辅助写代码、做页面
├── 分析业务场景，找出AI机会
├── 设计AI落地方案
└── 独立落地一个真实AI项目
```

### 认证层面

```
90天获得：
├── L1认证 - AI Builder操作能力
├── L2认证 - Vibe Coding能力
├── L3认证 - 场景设计能力
└── L4认证 - AI实战落地能力（最高）
```

### 成果层面

```
每个小组产出：
├── 1个可演示的AI工作流
├── 1份商业落地方案
└── 1个真实场景MVP（最小可行产品）
```

---

## 企业能得到什么？

| 投入 | 产出 |
|------|------|
| 90天培训投入 | 至少1个可上线的AI场景 |
| 人均学费 | 人效提升30-50%的AI工作流 |
| 导师时间 | 一支能自己落地AI项目的团队 |

---

## 一句话总结

> **90天后，企业有可用的AI系统，学员有可证明的能力认证。**
