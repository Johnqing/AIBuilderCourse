## 🎯 讲师守则

1. **以学员为中心**：让学员动手操作，而非单纯讲解
2. **及时反馈**：发现学员问题，第一时间解答
3. **鼓励提问**：没有蠢问题，只有不敢问的问题
4. **节奏把控**：根据学员反应调整教学速度
5. **记录复盘**：课后记录授课情况，持续优化
